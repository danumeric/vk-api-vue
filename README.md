Запуск
npm install
npm run serve
для запуска создайте файл .env в корневом каталоге и назначьте ваш токен ВК 
в переменную VUE_APP_TOKEN

Прод (с моим токеном) https://dan-api-vk.herokuapp.com/


Так как за один запрос можно получить количество друзей только одного польтзователя,
а количество запросов API ограничено сервером 3 запроса/секунду, количество друзей 
запрашивается с интервалом 400мс.
